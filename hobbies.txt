1. 형상관리 시작
   git init

2. 형상관리 상태 확인
   git status

3. 변경된 파일 스테이징 추가
   git add 파일명.확장자   
   git add . 

4. 형상관리 상태 확인
   git status

5. 새로운 버전 생성
   git commit -m "커밋 설명"
   git commit -m "게시판, 댓글 기능 추가"

6. 버전 확인
   git log
   git log --pretty=oneline
   git log --oneline

7. 버전 이동
   git checkout [버전]
   git checkout [브렌치]
   
8. 모든 작업을 되돌리고, 최신 버전으로 이동
   git reset --soft [버전] 또는 [브렌치]
   git reset --hard [버전] 또는 [브렌치]
